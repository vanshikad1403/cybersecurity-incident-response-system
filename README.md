# Cybersecurity Incident Response System

## 📌 Description
This project is a DBMS-based web application that models cybersecurity incidents, alerts, and mitigation actions.

## 🛠 Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MySQL

## 🔄 Features
- View Events and Alerts
- Add Mitigation Actions
- Full-stack integration
